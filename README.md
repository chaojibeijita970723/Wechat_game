# 无尽球场突围

一款足球主题的轻量级生存动作游戏。玩家通过摇杆走位，角色自动攻击，在不断增强的阵型中升级技能、击败精英和多阶段 Boss。

## 两个版本

### HTML 浏览器版

目录：`html-version/`

直接双击 `html-version/index.html` 即可在现代浏览器中运行。电脑端按住鼠标拖动控制移动，手机端直接触摸拖动。

如果浏览器限制本地音频，可在仓库根目录运行：

```bash
python -m http.server 8080
```

然后打开 `http://localhost:8080/html-version/`。

### 微信小程序版

目录：`wechat-miniprogram/`

在微信开发者工具中选择“导入项目”，项目目录指向 `wechat-miniprogram`。如果当前 AppID 无权使用，请在导入时替换为自己的 AppID 或测试号。

## 核心内容

- 四名拥有独立攻击方式和卡通形象的可选球员
- 摇杆走位、自动攻击和升级三选一
- 直线、包围圈、楔形等敌人阵型
- 五类特色敌人和三阶段 Boss
- 技能联动、赛场事件、自动拾取道具
- 连击、士气爆发、成就和角色熟练度

## 项目结构

```text
Wechat_game/
├─ html-version/          # 可直接浏览器运行
└─ wechat-miniprogram/    # 微信开发者工具工程
```

本项目为非商业游戏原型，未使用俱乐部徽标或官方球员照片。
